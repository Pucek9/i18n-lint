Usage

CLI

After installing, you should be able to type hslint into a terminal. {% highlight bash %} # Display version and exit $ hslint --version # Lint myfile.html $ hslint myfile.html # Lint all HTML files using a glob pattern $ hslint views/**/*.html # Set options using --

API

After installing, you can require the hslint lint module in your code. {% highlight javascript %} // Require the module var hslint = require('hslint'); // Lint myfile.html var errors = hslint('myfile.html'); // Lint all HTML files using a glob pattern var errors = hslint('views/**/*.html'); // Set options using an object as the second parameter var errors = hslint('views/**/*.html', { /*some options*/ }); {% endhighlight %}
