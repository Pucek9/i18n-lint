Usage

CLI

After installing, you should be able to type i18n-lint into a terminal. {% highlight bash %} # Display version and exit $ i18n-lint --version # Lint myfile.html $ i18n-lint myfile.html # Lint all HTML files using a glob pattern $ i18n-lint views/**/*.html # Set options using --

API

After installing, you can require the i18n-lint lint module in your code. {% highlight javascript %} // Require the module var i18nlint = require('i18n-lint'); // Lint myfile.html var errors = i18nlint('myfile.html'); // Lint all HTML files using a glob pattern var errors = i18nlint('views/**/*.html'); // Set options using an object as the second parameter var errors = i18nlint('views/**/*.html', { /*some options*/ }); {% endhighlight %}
