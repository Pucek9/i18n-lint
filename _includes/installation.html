Installation

CLI

Installing HSLint globally via npm gives you the hslint binary.

{% highlight bash %} $ npm install -g hslint {% endhighlight %} {% highlight bash %} $ hslint --help Usage: hslint [OPTIONS] Options: -h, --help output usage information -V, --version output the version number -a, --attributes Comma-separated list of HTML attributes to lint (default: 'alt,title') -i, --ignore-tags Comma-separated list of names of tags to ignore whilst linting (default: 'script,style') -t, --template-delimiters Template delimiters used in source files. For example, Mustache-like templating languages should use '{{,}}' -r, --reporter Specify which reporter to output results with --exclude Comma-separated list of glob patterns to ignore, e.g. "/test_subdir/,ignored.html" --color Force colored output --no-color Disable colored output Use `man hslint` for more information {% endhighlight %}

API

HSLint can be used in other projects as a library. Install it and then just require it:

{% highlight bash %} $ npm install --save hslint {% endhighlight %} {% highlight javascript %} var hslint = require('hslint'); // Lint a file var errors = hslint('myfile.html', { // ... options ... }); // Lint a string var errors = hslint.scan('
Untranslated String!
', { // ...options... }); {% endhighlight %}

Grunt Plugin

HSLint is also available as a grunt plugin: grunt-hslint.

{% highlight bash %} $ npm install --save-dev grunt-hslint {% endhighlight %} {% highlight javascript %} // Gruntfile.js grunt.loadNpmTasks('grunt-hslint'); grunt.initConfig({ // ... snip ... hslint: { // ... task config ... } // ... snip ... }); {% endhighlight %}
