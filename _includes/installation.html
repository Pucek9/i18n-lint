Installation

CLI

Installing HSLint globally via npm gives you the hslint binary.

{% highlight bash %} $ npm install -g hslint {% endhighlight %} {% highlight bash %} $ hslint --help Usage: hslint [OPTIONS] Options: -h, --help -V, --version -a, --attributes -i, --ignore-tags -t, --template-delimiters -r, --reporter {% endhighlight %}

API

HSLint can be used in other projects as a library. After installing, simply require the module.

{% highlight bash %} $ npm install --save hslint {% endhighlight %} {% highlight javascript %} var hslint = require('hslint'); hslint('myfile.html', { // ... options ... }); {% endhighlight %}

Grunt Plugin

Integrate HSLint with grunt and make it part of your build step using grunt-hslint.

{% highlight bash %} $ npm install --save-dev grunt-hslint {% endhighlight %} {% highlight javascript %} // Gruntfile.js grunt.loadNpmTasks('grunt-hslint'); grunt.initConfig({ // ... snip ... hslint: { // ... task config ... } // ... snip ... }); {% endhighlight %}
