Installation

CLI

Installing i18n-lint globally via npm gives you the i18n-lint binary.

{% highlight bash %} $ npm install -g i18n-lint {% endhighlight %} {% highlight bash %} $ i18n-lint --help Usage: i18n-lint [OPTIONS] Options: -h, --help -V, --version -a, --attributes -i, --ignore-tags -t, --template-delimiters -r, --reporter {% endhighlight %}

API

i18n-lint can be used in other projects as a library. After installing, simply require the module.

{% highlight bash %} $ npm install --save i18n-lint {% endhighlight %} {% highlight javascript %} var I18nLint = require('i18n-lint'); I18nLint('myfile.html', { // ... options ... }); {% endhighlight %}

Grunt Plugin

Integrate i18n-lint with grunt and make it part of your build step using grunt-i18nlint.

{% highlight bash %} $ npm install --save-dev grunt-i18nlint {% endhighlight %} {% highlight javascript %} // Gruntfile.js grunt.loadNpmTasks('grunt-i18nlint'); grunt.initConfig({ // ... snip ... i18nlint: { // ... task config ... } // ... snip ... }); {% endhighlight %}
